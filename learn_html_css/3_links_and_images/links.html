Links

Links are created using the "a" element which stands for anchor, and they look like this.

This page is about links! There are three types of links:

  • Absolute links like this one which links to the Mozilla Developer Network. Absolute links are the most detailed way to refer to a web resource. Typically they should be used only to refer to outside sources because hardcoding the path of all your own resources can lead to problems.
  • Relative links like a link to our own web page. Relative links are "relative" to the file currently being edited. They're nice but can get confusing.
  • Finally there are root-relative links which instead of being relative to the current page are relative to the root of the website. They won't work here, but they'd work on a web server. The most useful type of link. It would hypothetically look something like this.
