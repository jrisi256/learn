HTML Entities

There are three reserved characters in HTML: <, >, and &. Specifcally we invoke them using "HTML entities" which are invoked using an ampersand followed by the code and then ending with a semicolon. They're also useful for adding symbols not found on your keyboard like Æ. There are a lot of entities which can be found here.
