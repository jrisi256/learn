/Now where does the cascading part of style sheets come in? Well they're cascading because rules cascade down in the following way (ordered from least to most precedence):

  1. Browser default stylesheet
  2. User-defined stylesheets
  3. External stylesheets (that's us)
  4. Page-specific styles (also us but try to avoid)
  5. Inline styles (don't do this)

What are page-specific styles? Well you can use a special HTML element called the "style" element to define CSS rules specific to that page within the HTML page itself. Lives in the head because it's metadata. The problem is they're very hard to maintain.

What are inline styles? Well you can set the style "attribute" of any HTML element and then just stick the CSS there. Avoid this at all costs.

We can also, of course, link to multiple CSS stylesheets. Order matters! Ones that came later override ones which came earlier.
