Inline elements

We have have been using block-level elements up until now. Block-level elements are always drawn on a new line. In contrast, inline elements like this "em" tag affect sections of text within the line.

There is also the "strong" tag.

Last but not least we can emphasize and strengthen something by using both the "strong" and "em" tags (in any order so long as they're closed properly)

Empty elements

HTML condenses consecutive spaces into a single space when rendering in the browser. To manually insert different kinds of space, we use "empty" or "self-closing" elements.
